## 功能说明

转发数据到外部mqtt中，支持脚本
> 默认topic透传，

## 设计参数

|  参数   | 说明  | 必传  |默认值  |
|  ----  | ----  |----  |----  |
| ruleType  | 规则节点类型 |是 |MQTT  |
| script| 脚本 |是 |无  |


## yaml配置

   ```markdown
rules: # 规则引擎相关配置
    - ruleName: mqtt #名称
      chain: # 规则链
        - ruleType: MQTT # mqtt过滤器
          script: '{"topic":"${topic}","msg":"${msg}","qos":${qos}},"retain":false'
   ```
